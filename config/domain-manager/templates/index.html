Usage Instructions

Status Indicators

The Check column shows the real-time status of each record:

  • Pending: Record hasn't been validated yet or was recently modified.
  • Validating: Checking DNS records and Docker services.
  • Valid: DNS resolves correctly to this host and the service exists.
  • Invalid: There is an issue (DNS mismatch, service missing, etc.). Hover to see details.

Editing & Actions

  • Unsaved Changes: Rows with pending changes are highlighted in orange.
  • Service Selection: Search from running containers or use apache-host.
  • Anubis: Automatically verifies subdomain.root_domain DNS record.
  • Save Changes: Persists all records. All entries must be valid (Green) to save.
  • Restart Stack: Applies changes. Only required if saved data actually changed.
